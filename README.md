to run the project
==================
####have `python` (>=2.7), `pip` and `virtualenv` installed    


      $ git clone https://github.com/gdzhu/whatitwas.git
      $ cd db_final/mysite/src/
      $ virtualenv venv
      $ . venv/bin/activate
      $ pip install -r requirements.txt
      $ python manage.py syncdb
      $ python manage.py runserver
